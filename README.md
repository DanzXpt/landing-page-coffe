# Kopi Senja

Landing page coffee shop modern dengan tema hangat, elegan, dan responsive. Project ini dibuat menggunakan HTML, CSS, dan JavaScript vanilla.

## Fitur

- Responsive design untuk desktop dan mobile
- Navbar glassmorphism
- Dark mode dengan localStorage
- Menu kopi dengan filter kategori
- Search menu kopi
- Mini cart
- Quantity item di cart
- Total harga otomatis
- Checkout via WhatsApp
- Modal detail menu
- Toast notification
- Animasi scroll tanpa library
- Active navbar saat scroll
- Floating WhatsApp button
- Back to top button
- Gallery section
- Font Awesome icons

## Teknologi

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts
